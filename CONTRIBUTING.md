# Contribution and Branch Flow

## Branches
- `main`: stable releases. Merge only via approved PRs.
- `develop`: integration. Merge via PR from features and later to `main`.

## Naming
- Features: `feature/<short-name>`
- Fixes: `fix/<issue>`
- Docs: `docs/<topic>`
